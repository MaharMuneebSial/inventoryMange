# Complete Keyboard Shortcuts Guide

## Global Navigation Shortcuts

### Sidebar Navigation
- **Alt + 1-9** - Navigate to menu items (1=Dashboard, 2=New Product, 3=Sale, etc.)
- **Ctrl + D** - Open Dashboard
- **Ctrl + P** - Open Purchase page
- **Ctrl + S** - Open Sale page
- **Ctrl + L** - Logout

### Form Field Navigation
- **Tab** - Move to next field
- **Shift + Tab** - Move to previous field
- **Shift + Arrow Right** - Move to next field
- **Shift + Arrow Left** - Move to previous field
- **Enter** - Move to next field or submit form
- **Escape** - Close dropdowns/modals

### Table/List Navigation
- **Arrow Up/Down** - Navigate between rows
- **Shift + Arrow Up/Down** - Navigate between table rows
- **Page Up/Down** - Navigate pages in pagination
- **Home** - Go to first page
- **End** - Go to last page
- **Ctrl + F** - Focus search field

### Action Buttons
- **Ctrl + N** - New/Add button
- **Ctrl + E** - Edit selected item
- **Delete** - Delete selected item
- **Ctrl + Enter** - Submit/Save form
- **Escape** - Cancel/Close

## Page-Specific Shortcuts

### Dashboard
- **Tab** - Switch between Category, Sub-Category, Brand, Supplier, Unit tabs
- **Ctrl + 1-5** - Direct tab access (1=Category, 2=Sub-Category, etc.)

### Purchase/Sale Pages
- **F2** - Open packaging calculator
- **F3** - Focus on search product field
- **F4** - Add to list
- **F5** - Refresh/Clear form
- **Ctrl + S** - Save purchase/sale

### Product Entry
- **F1** - Focus product name
- **F2** - Focus category
- **F3** - Open product search
- **Ctrl + S** - Save product

### All List Pages (Sales List, Purchase List, Product Entry)
- **Ctrl + E** - Edit selected item
- **Ctrl + Delete** - Delete selected item
- **Ctrl + P** - Print
- **Ctrl + F** - Search/Filter

## Dropdown Navigation
- **Arrow Down** - Open dropdown or move to next item
- **Arrow Up** - Move to previous item in dropdown
- **Enter** - Select highlighted item
- **Escape** - Close dropdown
- **Type to search** - Filter dropdown items

## Modal/Dialog Navigation
- **Tab** - Move between buttons in modal
- **Enter** - Confirm action
- **Escape** - Cancel/Close modal
- **Arrow Left/Right** - Switch between Yes/No or OK/Cancel buttons
